## How to use this glossary

- Use `preferred` terms by default when they match the context.
- If a page needs a different term for contextual reasons, explain that in the PR that introduces the deviation.
- If a term is not settled yet, add it to `## Terms under discussion` instead of forcing a final answer.
- Keep rows short and focused so the glossary stays easy to review.
